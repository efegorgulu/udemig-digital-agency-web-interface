# Udemig Digital Agency Web Interface (HTML & CSS & SCSS)

A static **digital agency web interface** built with **HTML5, CSS3, and SCSS**, featuring a responsive navigation bar, hero section, services, portfolio, testimonials, and a modern UI layout.

## Screenshot
![](Screen.gif)

## Highlights
- **Responsive header/navigation** with mobile menu layout  
- **Hero section** with headline, description, and call-to-action buttons  
- **Services section** with icon-based cards  
- **About section** including mission, vision, and values  
- **Portfolio section** showcasing experience and capabilities  
- **Testimonials section** with client feedback and ratings  
- **Newsletter subscription UI** with email input field  
- **Footer section** with links, contact info, and social media icons  
- **Reusable button styles** with hover effects  
- **Responsive layout** using Flexbox and CSS Grid  

## Technologies Used
- **HTML5**
- **CSS3**
- **SCSS (Sass)**
- **Remix Icons**

## Installation
1. Clone or download this repository  
2. Open `index.html` in your preferred web browser  

### Optional (for developers)
If you want to edit SCSS files:

1. Install Sass:
   ```bash
   npm install -g sass
   ```
2. Compile SCSS to CSS:
   ```bash
   sass style.scss style.css
   ```

### Recommended (VS Code)
- Open the project in Visual Studio Code  
- Install a Sass extension (e.g., Live Sass Compiler)  
- Run the compiler to automatically generate CSS  

## Project Notes
- Front-end user interface only (**no backend or JavaScript functionality**)  
- Navigation menu button is **UI-only** (no toggle functionality)  
- SCSS is compiled into CSS — browsers do not support SCSS directly  
- Images are loaded from the `assets/` directory  
- Fully responsive design with modern layout techniques
